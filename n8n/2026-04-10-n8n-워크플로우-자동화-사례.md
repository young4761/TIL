### Context
n8n을 사용하여 간단한 워크플로우를 구축하는 과정에서, 디스코드 서버에서 메시지를 수신하면 자동으로 TIL 형식으로 변환하고 이를 GitHub에 푸시하는 작업을 설정했다. 이러한 자동화는 반복적인 작업을 줄이고, 목표에 맞춘 콘텐츠 생성에 신속하게 대응할 수 있도록 했다.

### Core
```json
{ "workflow": { "trigger": { "type": "Discord Message Received" }, "actions": [
   { "action": "Convert to TIL format" },
   { "action": "Push to GitHub" },
   { "action": "Update README.md" },
   { "action": "Send Completion Message" }
 ] }}
```

### Insight
n8n을 처음 사용해보면서 이 자동화가 얼마나 쉽게 설계될 수 있는지 깨달았다. 기본적인 노드의 입력과 출력을 이해하고, 그 흐름을 설정하는 것만으로도 복잡한 작업을 효율적으로 수행할 수 있다. 후츠릿의 조언대로, 가능하면 다양한 실습을 통해 그 이해도를 높이기 바란다.
